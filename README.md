# 不敢婚、不敢生：中国年轻人的婚育困局

这是一个面向移动端优化的静态数据报告页面，聚焦中国年轻人婚育行为变化、结构性原因与政策讨论。

## 在线访问

- GitHub Pages: <https://dingchenchen6.github.io/marriage-birth-report/>

## 项目结构

- `index.html`：网站入口页（GitHub Pages 默认入口）
- `marriage_birth_report_qr.png`：页面二维码（可选）
